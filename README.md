# AutoFilledIn

## 这是什么？

如你所见，这是一个用来辅助填写档案的工具

这个项目灵感来源于我被~~压榨~~的经历

知周所众，智慧共青团是一个**非常好用**的工具，可以让你在输入十数份甚至数十份大同小异的档案时反复被相同的数据折磨甚至做到——

*不看屏幕闭眼录入*

~~不要问我是怎么知道的~~

总之呢，在我第一次被团委抓去做~~苦力~~**义务劳动**时我就萌生了这种想法，简而言之就是略过相似的内容，然后自动将数据按照固定的格式使用模拟按键录入，以达到提升效率的目的
